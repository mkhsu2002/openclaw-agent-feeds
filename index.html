OpenClaw 龍蝦升級飼料包

讓你的 OpenClaw AI Agents 做事更準、更穩、更有紀律。

OpenClaw 官網 前往 GitHub Repo

什麼是「龍蝦飼料」？

本專案提供一系列預配置的 行為升級腳本 (Feeds)。您可以將其視為 OpenClaw AI Agents 的「行為準則指南」，藉由注入 防護決策迴圈 (Guarded Decision Loop) 與嚴謹的執行協議，它能大幅強化您的 AI Agents 能力：

  • 條件分流 (Conditional Branches)：賦予 AI Agents 釐清、重試與報錯的決斷力，取代盲目瞎猜。
  • 強制驗證 (Mandatory Verify)：建立產出即校驗的審查文化，大幅壓制各種幻覺。
  • 安全協議 (Safety Protocols)：在修改專案前強制 Inspect 與 Backup，提供反悔保護。
如何使用 Feeds？
  1. 在下方的「飼料包 Buffet」中挑選所需的擴充能力。
  2. 點擊卡片閱讀 技術規格詳情。
  3. 點擊 「安裝指南」，複製可直接貼給 AI Agents 的 Feed 指令。
  4. 臨時使用時，直接貼給 AI Agents；要讓規則常駐，再寫入工作區的 AGENTS.md。
