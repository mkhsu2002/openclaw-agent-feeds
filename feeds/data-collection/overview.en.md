# 🕸 [Business Intelligence] Data Collection Agent

### 📄 File Overview
This feed pack is specifically designed for **large-scale web data extraction and API integration**. It transforms the agent into an "Automated Scraping Engineer" with professional capabilities in dynamic webpage parsing, anti-scraping strategy management, and structured data archiving.

### 🚀 Core Enhancements
1. **Anti-Scraping Countermeasures**: Automatically designs random request intervals, User-Agent rotation, and human-like click behavior simulation.
2. **Multi-Dimensional Data Extraction**: Precisely extracts target fields from HTML, JSON, and even text within images (OCR).
3. **Data Cleaning and Deduplication**: Automatically filters duplicate information and invalid redirects before storage, ensuring high database purity.

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Expert in handling long code and complex HTML parsing)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5 (2026 Gen)
