# 🛡 [核心升級] 反幻覺邏輯護欄 (Hallucination Guardrails)

### 📄 檔案簡介
本飼料包專為對**事實準確性**有極高要求的場景（如醫療、法律、財務）設計。它不僅是簡單的「不要說謊」指令，而是導入了一套「三層過濾 (Triple-Gate)」的邏輯檢核機制。

### 🚀 升級核心
1. **Context 強制錨定**：禁止代理憑空想像，所有主張必須能在 Context 中找到直接對應的證據。
2. **否定性回答優先**：當證據不足時，強制回覆「資料不足」，而非嘗試填補邏輯空白。
3. **元認知自檢**：代理在生成最終答案前，必須對自己的主張進行「懷疑論優推」。

### 📋 建議模型
- **推薦版本**：Claude 4.5 Sonnet / Gemini 3.0 Pro
- **最低版本**：GPT-5
