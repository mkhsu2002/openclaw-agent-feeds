# 龍蝦通用核心飼料 (L0) - 概覽

這是所有 OpenClaw 代理（龍蝦）的基礎出廠設置。它定義了代理的核心身分、專業語調、標準思維邏輯與輸出規範，是一個「人人都可餵食」的通用基本款。

### 🌟 核心功能
- **身分確立**：明確定義代理作為 OpenClaw 生態系成員的角色。
- **思維鏈 (CoT)**：強制執行「先思考、後行動」的邏輯流程。
- **風格統一**：確保專業、簡潔且具啟發性的交流風格。
- **異常處理**：規範當指令模糊或超出範圍時的應對策略。

### 📊 適用對象
- 剛建立的全新 AI 代理。
- 需要標準化溝通介面的企業級助理。
- 任何希望提升 AI 回覆穩定性的使用者。

### 🚀 預期效果
餵食此飼料後，您的 AI 代理將展現出更高的專業度與邏輯連貫性，減少無效對話。
