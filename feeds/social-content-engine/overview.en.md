# ⚡️ [Automation] Social Content Engine

### 📄 File Overview
This feed pack is specifically designed for **rapidly increasing social media organic exposure and interaction**. It merges psychology, trend forecasting, and algorithm adaptation to automatically produce content modules with "viral" potential, covering text, storyboards, and title optimization.

### 🚀 Core Enhancements
1. **Hook Library**: Built-in 50+ verified opening templates, automatically adapted for different platforms.
2. **Emotional Trigger Mapping**: Automatically identifies and injects key emotional factors like "surprise, anger, joy, resonance" to increase sharing rates.
3. **SEO Tag Integration**: Ensures social content has better long-term traffic in search engines (e.g., Google, Threads search).

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Most flexible creativity and sense of language)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5 (2026 Gen)
