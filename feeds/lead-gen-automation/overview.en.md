# 🎯 [Automation] Lead Gen Automation

### 📄 File Overview
This feed pack is designed for **B2B development and high-value salespersons**. It transforms the agent into an "all-network development expert" capable of automatically identifying "buying signals" in LinkedIn, forums, and social media, and producing highly targeted Cold Reach messages.

### 🚀 Core Enhancements
1. **Intent Recognition Engine**: Automatically determines if a user's remark is "just asking" or possesses real "buying intent."
2. **Personalized Cold Outbound**: Rejects spammy mass-mailing code; every cold reach message is automatically customized based on the target's background.
3. **Follow-up Matrix**: Automatically plans non-intrusive follow-up logic within 7 days to increase response rates.

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Expert in processing large amounts of social data analysis)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5
