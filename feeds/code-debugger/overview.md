# 🔍 [開發者工具] 代碼除錯代理 (Code Debugger)

### 📄 檔案簡介
本飼料包專為**複雜 Bug 跳轉與系統性能優化**設計。它不僅是查找語法錯誤，更能深入邏輯底層，分析執行 trace，找出隱藏的邊際案例 (Edge Cases) 與內存洩漏問題。

### 🚀 升級核心
1. **假設驅動除錯 (Hypothesis-Driven Debugging)**：代理會先提出三個可能的 Bug 成因，並逐一設計驗證步驟。
2. **堆疊追蹤分析 (Stack Trace Analysis)**：自動解析錯誤日誌，定位真正出問題的根源檔案與行數。
3. **補丁安全驗證**：在產出修補代碼的同時，主動預測修補後可能產生的副作用。

### 📋 建議模型
- **推薦版本**：Gemini 3.0 Pro (擅長處理長上下文的 Bug 追蹤)
- **最低版本**：Claude 4.5 Sonnet
