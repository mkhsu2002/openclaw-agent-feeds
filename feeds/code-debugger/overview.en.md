# 🔍 [Developer Tool] Code Debugger Agent

### 📄 File Overview
This feed pack is designed for **complex bug tracking and system performance optimization**. It goes beyond finding syntax errors to delve into the logic layer, analyzing execution traces to identify hidden edge cases and memory leak issues.

### 🚀 Core Enhancements
1. **Hypothesis-Driven Debugging**: The agent first proposes three possible bug causes and designs verification steps for each.
2. **Stack Trace Analysis**: Automatically parses error logs to locate the root cause file and line number.
3. **Patch Safety Verification**: Proactively predicts potential side effects while producing fix code.

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Expert in handling long-context bug tracking)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5
