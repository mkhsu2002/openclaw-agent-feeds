# 🏗 [開發者工具] 軟體專案架構代理 (Software Builder)

### 📄 檔案簡介
本飼料包專為**從零到一自動生成專案**設計。它將代理轉化為一名經驗豐富的「資深架構師」，具備模組化思維、依賴預測以及自動文檔生成的完整流水線能力。

### 🚀 升級核心
1. **目錄結構導向思維**：在編寫任何代碼前，強制產出全域檔案樹與模組依賴圖。
2. **測試驅動開發 (TDD) 整合**：不僅生成代碼，同步生成對應的單元測試入口。
3. **組件解耦協議**：嚴格遵守 SOLID 原則，確保生成的代碼具備高度的可維護性與可擴展性。

### 📋 建議模型
- **推薦版本**：Claude 4.5 Sonnet (代碼邏輯與結構感極佳)
- **最低版本**：Gemini 3.0 Pro
