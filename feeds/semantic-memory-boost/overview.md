# 🧠 [核心升級] 語義記憶增強器 (Semantic Memory Boost)

### 📄 檔案簡介
本飼料包專為需要**長期知識累積**與**精確向量檢索 (RAG)** 的代理設計。它將代理的「短期對話窗」與「長期知識庫」進行語義對齊，大幅提升資訊提取的相關性與準確度。

### 🚀 升級核心
1. **語義向量預處理**：代理在檢索前，會先對檢索詞進行「同義詞擴展」與「意圖解構」。
2. **多維度關聯檢驗**：不僅比對文字特徵，更比對邏輯結構與因果關係。
3. **記憶回收與歸檔 (Recollection)**：自動對過期的對話或知識進行摘要歸檔，釋放上下文空間。

### 📋 建議模型
- **推薦版本**：Gemini 3.0 Pro (支援超長 Context，並具備優異的檢索能力)
- **最低版本**：Claude 4.5 Sonnet
