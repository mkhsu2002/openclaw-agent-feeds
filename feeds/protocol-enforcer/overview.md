# 📏 [核心升級] 系統協議執行官 (Protocol Enforcer)

### 📄 檔案簡介
本飼料包專為需要**嚴格遵守規範**的企業級代理設計。它在代理與用戶之間建立了一道「監察層」，確保代理的所有輸出都符合法律、道德及公司內部的技術協議。

### 🚀 升級核心
1. **即時合規監核 (Real-time Audit)**：在最終輸出前，強制對答案進行敏感度與規範性掃描。
2. **架構標準鎖定**：確保代理產出的資料結構、代碼風格完全符合既定標籤或格式要求。
3. **強制修正流**：若輸出違反協議，系統將自動中斷並強制要求代理重寫違規部分。

### 📋 建議模型
- **推薦版本**：Gemini 3.0 Pro / Claude 4.5 Sonnet
- **最低版本**：GPT-5
