# 🦞 [Multimodal] Visual Auditor

### 📄 File Overview
This feed pack is designed for AI agents with **multimodal vision capabilities**. It simulates the eye of a human designer, performing pixel-level "compliance" and "aesthetic" audits on webpage screenshots, UI components, or graphic designs.

### 🚀 Core Enhancements
1. **Pixel-Level Scanning**: Identifies misalignments, insufficient contrast, or font errors.
2. **Design System Validation**: Checks for compliance with Brand Guidelines.
3. **Accessibility Review**: Automatically evaluates WCAG benchmark compliance.

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Strong spatial resolution capabilities)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5 (2026 Gen)
