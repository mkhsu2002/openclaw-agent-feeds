# 🧪 [Developer Tool] Test Suite Architect

### 📄 File Overview
This feed pack is designed for software projects with **extremely high stability requirements**. It goes beyond writing tests to building a complete testing boundary, automatically identifying edge cases, and producing a comprehensive test suite covering unit, integration, and E2E tests.

### 🚀 Core Enhancements
1. **Corner Case Mining**: Automatically derives extreme input values that might cause system crashes.
2. **Mocking Strategy Optimization**: Automatically designs simulation logic for external APIs and databases to achieve complete isolated testing.
3. **Coverage-Oriented**: Precisely generates test patches for untested logic branches.

### 📋 Recommended Models
- **Recommended**: GPT-5.3 / Gemini 3.0 Pro / Claude Sonnet 4.6 (Expert in logical reasoning and boundary testing)
- **Minimum**: GPT-5.1 / Gemini 3.1 Flash / Claude Haiku 4.5
