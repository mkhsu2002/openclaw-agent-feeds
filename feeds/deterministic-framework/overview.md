# 🦞 [核心升級] 龍蝦確定性代理框架 (Lobster Deterministic Framework)

### 📄 檔案簡介
本飼料包專為需要**極高輸出穩定性**與**邏輯嚴密性**的 AI 代理（Agents）設計。透過建立「心理護欄」與「狀態機邏輯」，將 LLM 的不確定性降至最低。

### 🚀 升級核心
1. **邏輯閘 (Logic Gates)**：強制代理在輸出前進行自檢。
2. **XML 結構化封裝**：使用 XML 標籤隔離思考過程與最終輸出。
3. **零幻覺約束**：嚴格限定代理僅能從提供的 Context 中擷取資訊。

### 📋 建議模型
- **推薦版本**：Gemini 3.0 Pro / Claude 4.5 Sonnet
- **最低版本**：Gemini 2.5 Flash / o1-mini (2026 Gen)
