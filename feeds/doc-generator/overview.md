# ✍️ [開發者工具] 文檔自動生成器 (Doc Generator)

### 📄 檔案簡介
本飼料包專為**減少開發者文檔負擔**設計。它能精準解析源代碼邏輯，自動生成 API 參考、系統架構圖以及安裝手冊。它不只是註釋翻譯，更是代碼意圖的深度轉譯。

### 🚀 升級核心
1. **依賴關係自動映射**：自動標註各模組間的調用關係。
2. **多格式輸出支援**：支援 Markdown, Swagger/OpenAPI, 以及 Mermaid 流程圖。
3. **技術債偵測**：在生成文檔時，同步標註代碼中缺少註釋或邏輯複雜的部分。

### 📋 建議模型
- **推薦版本**：Claude 4.5 Sonnet (文字表達極為精準)
- **最低版本**：Gemini 3.0 Pro
